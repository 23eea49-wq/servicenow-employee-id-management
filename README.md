# Employee ID Card Request Management System

## Project Overview

This project is developed in ServiceNow Studio to automate the Employee ID Card Request process.

Employees can submit requests for:

- New Employee ID Card
- Replacement ID Card
- Lost ID Card
- Damaged ID Card

The application automatically routes requests for manager approval, creates tasks for the admin team, and sends notifications after approval.

---

## Features

### Catalog Item

Employee ID Card Request Form

Fields:

- Employee Name
- Employee ID
- Department
- Request Type
- Reason

---

### Approval Workflow

1. Employee submits request
2. Manager receives approval request
3. Manager approves request
4. Admin task is created automatically
5. Notification is sent to employee
